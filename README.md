# NodeJs Token

>   - [Environment](#environment)
>   - [Dependencies](#dependencies)
>   - [Run](#run)
> - [License](#license)

## Usage
### Environment
Installation
```bash
$ npm install npm -g
```

### Dependencies
Install dependencies and modules
```bash
$ npm install
```

### Run
Compile project
```bash
$ npm start
$ npm run nodemon
```

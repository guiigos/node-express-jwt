# NodeJs Token
> Projeto de exemplo da utilização de **autenticação via token** utilizando [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)

### Instalar as dependências
```sh
$ npm install
```

### Executar
```sh
$ node server.js
```

ou

```sh
$ npm start
```
